# Tic-Tac-Toe
# 🎮 Tic-Tac-Toe AI

A modern and interactive **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**. The project offers both **Player vs AI** and **Local PvP** modes with multiple AI difficulty levels, a live scoreboard, sound controls, and a responsive modern UI.

---

## ✨ Features

### 🎮 Game Modes
- 🤖 Player vs AI
- 👥 Local Player vs Player

### 🧠 AI Opponent
- 🟢 Easy Mode
- 🟡 Medium Mode
- 🔴 Unbeatable Mode (Minimax AI)

### 🏆 Gameplay
- Automatic Winner Detection
- Draw Detection
- Turn Indicator
- Play Again Option
- Reset Scores
- Starting Player Selection (You / AI)

### 📊 Scoreboard
- Human Score
- AI Score
- Draw Counter
- Persistent Score Tracking During Session

### 🔊 User Experience
- Sound Effects Toggle
- Responsive Layout
- Smooth Animations
- Modern Dark UI
- Interactive Buttons
- Clean Game Interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ashmit-codebase/Tic-Tac-Toe.git
```

### Open the project

Simply open **index.html** in your browser.

No installation required.

---

## 🎯 Game Controls

- Choose **Vs AI** or **Local PvP**
- Select AI Difficulty (Easy / Medium / Unbeatable)
- Choose who starts first
- Click any cell to make your move
- Use **Play Again** to start a new match
- Use **Reset Scores** to clear the scoreboard
